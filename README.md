# express-prisma-api
REST API with ExpressJS and Prisma io
REST API apalication with NodeJS , ExpressJS and Prisma ORM. 


## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```
    
## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
